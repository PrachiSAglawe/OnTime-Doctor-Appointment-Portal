# OnTime – Doctor's Appointment Portal

![ASP.NET Core](https://img.shields.io/badge/ASP.NET%20Core-8.0-blue)
![C#](https://img.shields.io/badge/C%23-.NET-purple)
![SQL Server](https://img.shields.io/badge/SQL%20Server-2019-red)
![Bootstrap](https://img.shields.io/badge/Bootstrap-5-blueviolet)

A full-stack healthcare appointment management web application built using **ASP.NET Core MVC 8.0**.

## 🚀 Features
- 🔐 Role-based access for Patients, Doctors, and Administrators
- 📅 Appointment booking and management system
- 🗄️ CRUD operations using Entity Framework Core and LINQ
- 📱 Responsive UI built with Bootstrap and jQuery
- 🏗️ Repository Pattern and Dependency Injection

## 🛠️ Tech Stack
- Backend: ASP.NET Core MVC 8.0, C#
- ORM: Entity Framework Core (Code First)
- Database: SQL Server
- Frontend: HTML5, CSS3, Bootstrap, jQuery
- Tools: Visual Studio 2022, Postman, Swagger
